## 📖 코스 개요

Google Cloud 기반 데이터 분석의 기본 개념부터 BigQuery, Looker, Looker Studio를 활용한 실무 분석·시각화까지 다루는 입문 과정입니다.

### 🎯 학습 목표

- 데이터 분석 워크플로 및 분석 유형 이해
- Google Cloud 데이터 분석 제품군 파악
- BigQuery로 데이터 인사이트 도출
- Looker / Looker Studio로 시각화 및 의사결정 지원
